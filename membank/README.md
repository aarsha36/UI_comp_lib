This is a creative React UI project called **Memory Bank**, designed with a vintage film-roll and polaroid theme.

The project focuses only on **frontend UI design** (no backend functionality).

## Features
- Welcome screen with film-roll animation  
- Timeline of memories displayed as film frames  
- Clickable memory frames opening in a camera-style view  
- Favorite memory option  
- Snippets carousel with forward/back navigation
- add new memory option to add memeories
- profile option with user details,activity,places visited,places to visit and other options
- settings with standard setting options

## Technologies Used
- React.js  
- Bootstrap  
- Lucide Icons  
- CSS Animations

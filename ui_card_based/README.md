# Bootstrap Card Layout Project

This is a simple React project demonstrating a **card-based layout using Bootstrap**.

## Features
- Responsive grid of cards  
- Hover animation effect on cards  
- Bootstrap styled buttons  
- Clean and centered UI design  

## Technologies Used
- React.js  
- Bootstrap  
- Inline CSS Styling
